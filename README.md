# 🌾 王者农场专业务农 - 智能护肝版

本项目是对 [gzchenjin2/hokfarming](https://github.com/gzchenjin2/hokfarming) 开源项目的克隆与深度优化版本。

在保留了原版优秀的 UI 交互、数据中心与账号管家的基础上，重构了核心排期器，引入了**纯净状态机引擎**，大幅度提升了农场防偷菜博弈和个人作息的适配度。

## 🚀 核心升级特性

### 1. 📅 智能护肝排期器 (重构)
废弃了原版基于静态数学公式的预测模型，引入了底层**“1分钟步进式”时间沙盘状态机**。
* **🌙 护肝模式**：支持设定个人作息时间（入睡与醒来）。排期器会自动在你的梦境期间智能穿插“睡前强行补水”动作。
* **⚖️ 干旱顺延算法**：如果睡眠期间水管干涸，引擎将以极其精确的 `剩余总寿命扣减算法` 计算植物进度，哪怕睡懒觉，也只会造成极小的时间顺延。
* **🚨 深度睡眠防坑拦截**：如果你种下的作物，其最终的极限秒熟时间不幸落在了你设定的“深度睡眠期”（入睡后1小时至醒来前1小时），系统会直接弹出红框警告，阻止你成为半夜定闹钟的怨种。

### 2. 🤫 防偷菜憋气逻辑 (新增)
全网独家“防被偷博弈算法”实装。
* **CD 门槛探测**：浇水操作有 10% 的冷却 CD。引擎会在你需要补水时进行预判：如果浇水后剩余的成熟时间小于 CD 冷却时间，会导致你无法手动触发“秒熟斩杀”从而被偷菜。
* **强制憋气**：一旦触发上述条件，计算器会阻止你浇水，并在时间轴上生成一个带警示的 **"🤫 憋气等斩杀"** 节点。让植物干着，直到时间流逝刚好足够一刀秒熟。把主动权死死捏在自己手里！

### 3. ⏱️ 经典计算器 (完美保留)
* 保留了原版极其好用的、基于经典数学公式的“极限收割计算器”。
* 实时给出 `建议自然成熟`、`卡门槛警告` 或是 `终极秒熟倒计时`。

### 4. 📋 农场账号管家 (增强兼容)
* 保留原版账号管家及本地 localStorage 数据存储。
* 保留 CSV 格式的导入导出数据迁移功能。
* 保留滑动复选、下拉框一键从计算器/排期器同步数据等便捷操作。

## 🛠️ 如何使用

1. 下载或 Clone 本仓库代码。
2. 这是一个纯静态前端项目，无需任何服务端环境。
3. 直接在浏览器中双击打开 `index.html` 即可使用。
4. 建议在手机浏览器中添加到主屏幕，或使用作者原版的 APK 打包方案获得类原生体验。
